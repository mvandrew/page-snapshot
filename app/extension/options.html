Page Snapshot Settings

Настройте endpoint для отправки снимков страниц. Расширение будет отправлять HTML и метаданные страницы в формате JSON на указанный URL. Поддерживаются HTTP и HTTPS соединения.

Конфигурация Endpoint

Endpoint должен принимать HTTP/HTTPS POST запросы с JSON телом. Поддерживаются как HTTP, так и HTTPS соединения (включая самоподписные сертификаты).
